# ClickByChris Setup Tool 🚀

**Version : 1.0.0** | Juin 2026

Outil Windows permettant de préparer rapidement un PC avec une interface simple, moderne et centralisée.

## ✨ Features Principales

- ✅ **Détection multi-écran** (1920x1080 à 4K)
- ✅ **Scaling responsive** automatique
- ✅ **Installation d'applications** via winget
- ✅ **Composants de compatibilité** (Visual C++, .NET, WebView2)
- ✅ **Optimisations Windows** safe et stable
- ✅ **Structure de dossiers** automatisée
- ✅ **Raccourcis intelligents** personnalisés
- ✅ **Lecteur musique** intégré avec album art
- ✅ **Points de restauration** avant modifications
- ✅ **Logs détaillés** et rapports CSV
