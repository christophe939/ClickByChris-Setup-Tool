## 🎯 Avant de Poser une Question

### 📖 **Vérifie d'abord :**

1. **Lis la FAQ** → Peut-être que ta question est déjà répondue !
2. **Cherche dans les threads** → Quelqu'un d'autre a peut-être eu le même problème
3. **Vérifie la doc** → La réponse est peut-être dans [TROUBLESHOOTING.md](./TROUBLESHOOTING.md)

### ❓ **Ensuite, pose ta question :**

Dis-nous :
- 🔧 Qu'est-ce que tu as essayé de faire ?
- ❌ Qu'est-ce qui ne marche pas ?
- 💻 Quel Windows utilises-tu ?
- 📝 Quel est le message d'erreur ?

**Exemple :**

Salut ! J'ai un problème :

    Je lance ClickByChris v1.0.0
    L'interface ne s'affiche pas
    Windows 11 Pro
    Erreur : "Could not load file or assembly"

J'ai déjà essayé :

    Relancer le script
    Redémarrer mon PC
