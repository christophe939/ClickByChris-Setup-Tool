# 📖 Installation Guide - ClickByChris Setup Tool

## 🚀 Installation Rapide

### ✅ Prérequis

- Windows 10 (Build 19041+) ou Windows 11
- PowerShell 5.1 minimum (7.0+ recommandé)
- Droits administrateur
- Connexion internet (pour winget)

---

## 📥 Méthode 1 : Depuis la Release (RECOMMANDÉE)

### Étape 1️⃣ : Télécharger
1. Va sur [**Releases**](https://github.com/christophe939/ClickByChris-Setup-Tool/releases)
2. Télécharge **`ClickByChris-Setup-Tool-v1.0.0.zip`**

### Étape 2️⃣ : Extraire
1. Clique droit sur le ZIP
2. **"Extraire tout..."**
3. Choisis un dossier (ex: `C:\Tools\`)

### Étape 3️⃣ : Lancer
1. Ouvre le dossier extrait
2. **Double-clique** sur `Launch_ClickByChris.cmd`
3. Accepte l'UAC (message de sécurité Windows)
4. **Suis le wizard !** 🎯
